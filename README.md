# Описание проекта    
В репозитории представлен проект дашборда для проведения RFM-анализа клиентов компании Adventure Works Cycles с использованием возможностей Power BI. Основной целью проекта является сегментация клиентской базы на основе показателей их активности и финансовой отдачи для оптимизации маркетинговых стратегий. Такой подход позволяет перейти от массовых и затратных рекламных рассылок к точечным, персонализированным коммуникациям. В результате бизнес получает наглядный инструмент для эффективного планирования рекламных кампаний и повышения удержания клиентов.

    
# Описание данных
В качестве источника информации используется база данных AdventureWorks, содержащая агрегированные сведения о клиентах, продуктах и продажах компании за 2020–2022 годы. Для анализа формируется отдельная таблица уникальных клиентов, которая обогащается вычисляемыми столбцами: дата последней покупки, общее количество транзакций и суммарная выручка. Дополнительно рассчитывается показатель давности последней покупки в днях относительно максимальной даты в наборе данных для корректной оценки. Эти метрики служат надежным фундаментом для объективного анализа поведения и ценности каждой группы покупателей.


# Подход:
Подход включает полный цикл разработки: от ETL-процессов и построения связей в модели данных до написания пользовательских мер на языке DAX. Сегментация выполняется по трем ключевым параметрам (Recency, Frequency, Monetary) с использованием процентилей для разделения клиентов на группы от лучших к худшим. На основе комбинации этих оценок формируется итоговый RFM-код, который затем маппится на укрупненные смысловые категории, такие как «VIP», «Выгодные», «Спящие» и «Потерянные». Визуализация реализуется через диаграмму дерева, вложенные матрицы и карточки с ключевыми показателями для наглядного отображения долей и вкладов каждого сегмента.

# Выводы и результаты: 
Проведенный анализ позволяет выявить ключевые паттерны поведения клиентов, например, преобладание разовых покупателей или наличие узкой, но высокодоходной группы лояльных клиентов. Результаты дашборда дают маркетинговой команде четкое понимание того, какие сегменты требуют срочных программ по возврату, а какие готовы к дополнительным продажам. Внедрение такого инструмента снижает затраты на неэффективные рекламные кампании за счет таргетированного взаимодействия с аудиторией. В конечном итоге, использование RFM-сегментации способствует повышению общей лояльности клиентов и росту долгосрочной прибыли компании.
