# Описание проекта    
В данном репозитории представлен дашборд, демонстрирующий мой подход к составлению дашборда (подложка, компановка макета, ETL, меры DAX). 
В рамках текущего проекта проведён анализ продаж и клиентов для распределения последних по сегментам в зависимости от показателей их активности и выручки. Это позволит в дальнейшем эффективно спланировать рекламные кампании и прочие маркетинговые мероприятия. 

    
# Описание данных
Adventure Works Cycles - крупная многонациональная производственная компания, которая осуществляет производство и продажу велосипедов на рынках Северной Америки, Европы и Азии. Штаб-квартира Adventure Works Cycles находится в Ботелле, штат Вашингтон, где в компании работает 500 человек. Кроме того, в Adventure Works Cycles работает несколько региональных отделов продаж.

База данных AdventureWorks первоначально была опубликована корпорацией Майкрософт, чтобы показать, как создать базу данных SQL Server с использованием SQL Server 2008. База данных поддерживает вымышленную компанию под названием Adventure Works Cycles. В БД представлены данные о клиентах, продуктах и продажахкомпании за 2020-2022 гг.
